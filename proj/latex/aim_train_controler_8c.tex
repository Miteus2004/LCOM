This module controls the gameplay logic for an aiming training program. It includes functions for starting different game modes such as Time Attack and Classic, handling target hits and misses, updating scores and lives, and managing the display of the game interface. The State Controller module is responsible for managing the overall state and flow of the game. It provides functions to initialize different game states, such as displaying the main menu, starting the game of testing the reaction speed, aiming, handling game over scenarios, and displaying the scoreboard. Each function within the module transitions the game to a specific state and ensures proper initialization by invoking necessary setup functions.
